# @beam/extension-protocol

This package is the future generated TypeScript home for Beam's native extension runtime protocol.

It is intentionally introduced before code generation is wired up so the workspace structure and ownership are set early.

Planned responsibilities:

- generated TypeScript bindings from `proto/extension-runtime`
- shared runtime message types for Beam SDK and extension manager
- one place to version the native extension contract

Current state:

- scaffold only
- no generated code yet
- no runtime consumers yet
