# Beam Web

Landing page for Beam, the Linux-first command launcher.

## Stack

- Astro 6
- React 19 islands
- Tailwind CSS 4
- Framer Motion

## Development

```sh
bun install
bun run dev
```

## Build

```sh
bun run build
```

## Notes

- The live route is [`src/pages/index.astro`](src/pages/index.astro).
- Shared page metadata lives in [`src/layouts/Layout.astro`](src/layouts/Layout.astro).
