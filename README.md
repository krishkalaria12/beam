# beam

minimal launcher-style desktop app built with tauri + react.

## stack

- tauri v2 (rust backend)
- react + tanstack router (frontend)
- vite + bun (tooling)
- tailwind css v4 (styling)

## run locally

```bash
bun install
bun run desktop:dev
```

## build desktop app

```bash
bun run desktop:build
```

## current ui

- initial launcher shell with a centered panel
- heading only: `beam`
